### Инициализация репозитория

### Глобальная конфигурация

Имя пользователя:
`git config --global user.name 'Fedotov'`

`git config --global user.email 'kemt.info@gmail.com'`

`git remote add origin https://github.com/kemtinfo/soft-lab.git`
